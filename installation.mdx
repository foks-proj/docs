## Managed Installers

The easiest way to install FOKS is via a platform package.

<Tabs>
  <Tab title="macOS">
    Download the macOS installer from [foks.pub](https://foks.pub) or install via Homebrew (coming soon).

    After installation, create the git remote helper symlink (needed once):

    ```bash
    make git-link
    # or manually:
    ln -s $(which foks) $(dirname $(which foks))/git-remote-foks
    ```
  </Tab>
  <Tab title="Debian / Ubuntu">
    ```bash
    # Add the FOKS apt repository, then:
    sudo apt-get install foks
    ```

    You also need the PCSC lite library for YubiKey support:

    ```bash
    sudo apt-get install libpcsclite-dev pcscd
    ```
  </Tab>
  <Tab title="Fedora">
    ```bash
    sudo dnf install foks
    ```
  </Tab>
  <Tab title="Arch Linux">
    ```bash
    yay -S foks
    ```
  </Tab>
  <Tab title="Windows">
    Download the Windows installer from [foks.pub](https://foks.pub).

    On Windows, `git-remote-foks` is a copy of (not a symlink to) the `foks.exe` binary.
  </Tab>
</Tabs>

## Build from Source

Requires Go 1.24 or later.

```bash
git clone https://github.com/foks-proj/go-foks.git
cd go-foks
make             # builds for the local platform, installs into $GOPATH/bin
make git-link    # creates the git-remote-foks symlink (needed once)
```

### Linux prerequisites

On Linux you need the PCSC lite development library before building:

```bash
sudo apt-get install gcc make libpcsclite-dev pkg-config
```

For YubiKey support at runtime:

```bash
sudo apt-get install pcscd
```

If your distribution restricts access to `pcscd` for non-root users, add a polkit rule:

```bash
cat <<EOF > /etc/polkit-1/rules.d/90-pcscd.rules
polkit.addRule(function(action, subject) {
    if (action.id == "org.debian.pcsc-lite.access_pcsc" &&
        subject.isInGroup("pcscd")) {
        return polkit.Result.YES;
    }
});
EOF

groupadd pcscd
usermod -aG pcscd $USER
systemctl daemon-reload
systemctl restart polkit
systemctl restart pcscd
```
